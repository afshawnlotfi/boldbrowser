# boldbrowser
A Browser Made in Swift

This browser satisfied a need to do better multitasking on iOS devices that allowed for increased productivity.

# Features
* Download Web Pages
* Better Controls
* Favicons
* Better Tab Management
* Hashtag Workspaces
* Nice plugin framework
* History Management
